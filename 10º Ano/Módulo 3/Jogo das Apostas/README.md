# Módulo 3 - Jogo das Apostas

Pretende-se um programa que permita implementar um jogo de apostas a partir de uma interface gráfica.

Descrição do programa:

1. O jogo começa quando o utilizador clicar no botão “Jogar”, depois de ter indicado o nome do jogador e o número de fichas com que vai jogar;
2. Em cada aposta, o jogador deve indicar previamente quantas fichas quer apostar e em que número do dado aposta. Se indicar “Todas as fichas” aposta todas as fichas que tem.
3. A clicar no botão “Apostar”, o programa deve gerar aleatoriamente um número entre 1 e 6, correspondente aos números de um dado virtual.
4. Se o jogador acertou no número do dado (o número apostado = número do dado) o jogador ganha o dobro das fichas que apostou. Senão, perde as fichas que apostou.
5. Em cada aposta, o campo do formulário “Nº de Fichas” deve ser actualizado com mais ou menos fichas conforme o jogador ganhou ou perdeu. Paralelamente, este deve ser informado, através das labels criadas para o efeito, do número do dado que saiu e se perdeu ou ganhou.
6. O botão “Fim do jogo” permite fechar a janela e terminar o programa.
7. Atenção: o jogador não pode apostar mais fichas do que aquelas que possui.
