# Módulo 10 Exercício 2 - Heranças

Crie um programa TestaFilmes.
 Crie uma classe Filme com os atributos título, ano, realizador, tipo, duração. Encapsule os atributos.
Construa as classes Filme estrangeiro e EdicaoDoRealizador que herdam da classe Filme.  
A classe FilmeEstangeiro deve ter informação sobre o país de origem e a língua em que o filme é falado.  
A classe EdicaoDoRealizador deve ter informação (uma string) sobre as alterações feitas ao original.
No programa crie um filme, um filme estrangeiro e um filme EdicaodoRealizador, visualize os dados de cada um.
