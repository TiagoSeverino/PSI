# Módulo 9 - Ficha de avaliação

Elabora um programa em modo consola que permita inserir os dados de uma corrida de automóveis. Dê-lhe o nome de TesteCorrida.

1. Crie uma classe Corrida.

1.1 A classe tem como atributos o número da corrida, o local onde é realizada,a data em que é realizada, o número máximo de carros que podem participar e o número de carros a participar. Encapsule os atributos.

1.2 Crie um método que permita adicionar um carro aos carros a participar. Verifique se o número máximo de carros foi atingido e mostre uma mensagem de erro caso aconteça.

2.  Crie uma classe piloto com os atributos nome, idade, sexo, peso e altura.Encapsule os atributos.

3.Crie uma classe Carro, com os atributos marca, modelo e piloto. Encapsule os atributos. Crie um método que mostre no ecrã todos os atributos do carro.

4. Crie uma classe equipa com o atributo nome. Cada equipa é constituída obrigatoriamente por 3 carros. Os carros são adicionados à equipa no construtor.

5. Crie uma classe Pontuacoes com os atributos corrida, equipa, pontuacao. Encapsule os atributos. Crie um método que permita visualizar o número da corrida, a data, o local, o nº de carros participantes e a equipa vencedora.

6. No programa principal crie uma nova corrida. Crie nove pilotos. Crie nove carros. Adicione os carros à corrida. Crie três equipas.  Visualize os dados das pontuações.

Nota: Adicione as variáveis e os métodos que achar necessário.
