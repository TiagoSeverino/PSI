# Módulo 9 - Projeto de avaliação do módulo 9.

Pretende-se que crie um programa em C# que registe determinados  dados, produtos ou serviços semelhante ao realizado no exercício 8.

Podem escolher a atividade que quiserem, por exemplo:

registo do stock de uma loja
registo de animais de um veterinário
registo de automóveis numa oficina ou stand de automóveis
registo de nascimentos num determinado hospital, etc..
Devem respeitar as seguintes especificações:

O projeto terá que estar baseado em classes com atributos e métodos;
Todos os atributos devem ser encapsulados;
Valide o preenchimento dos campos;
Deve ser possível realizar as quatro operações básicas de um formulário; inserir, consultar, alterar e eliminar registos. 
Não serão aceites dois trabalhos idênticos;
O projeto é para ser entregue no dia 15 de Novembro.
Bom trabalho!
