# Ficha de trabalho 2
Elabore um programa que contemple as seguintes informações utilizando memória

dinâmica:

a.- Crie uma estrutura (registo) que permita registar o nome, o ano da disciplina e a classificação de um aluno;

b.- Crie um procedimento para criar uma PILHA de dados vazia;

c.- Crie uma função para verificar se a pilha de dados está, ou não, vazia;

d.- Crie um procedimento para inserir um novo registo na pilha;

e.- Crie um procedimento para remover um elemento da pilha;

f.- Crie uma função (com um parâmetro) que permita verificar a classificação de um dado aluno a inserir pelo utilizador;

g.- Elabore um menu que permita chamar as funções e procedimentos acima mencionados, de forma a inserir as várias classificações de cada aluno de acordo com o numero de alunos assinalados pelo utilizador.
