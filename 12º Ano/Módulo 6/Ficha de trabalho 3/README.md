# Ficha de trabalho 3
Elabore um programa que contemple as seguintes informações utilizando memória dinâmica:

a. Crie uma estrutura que permita registar o nome de uma cidade (máx. 20 caracteres) e as temperaturas médias (ao longo de todos os meses do ano) do tipo inteiro;

b. Crie um procedimento para criar uma FILA de dados vazia;

c. Crie uma função para verificar se a FILA de dados está, ou não, vazia;

d. Crie um procedimento para inserir um novo registo na FILA;

e. Crie um procedimento para remover um elemento da FILA;

f. Crie uma função que permita verificar a temperatura máxima do ano registado;

g. Crie uma função/procedimento que permita verificar qual a cidade e o mês onde se obteve a maior temperatura do ano;

h. Elabore um menu que permita chamar as funções e procedimentos acima mencionados, de forma a inserir as várias temperaturas de várias cidades de acordo com o numero de cidades assinalados pelo utilizador.
