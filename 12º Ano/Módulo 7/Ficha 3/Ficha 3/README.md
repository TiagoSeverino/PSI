# Ficha 3 - Agenda
Envia para avaliação o programa que implemente uma agenda de contactos. A gestão deve ser feita com estruturas dinâmicas, devendo os dados serem registados e carregados a partir de um ficheiro de dados estruturado.

Cada contacto deve conter a seguinte informação: nome, telemóvel, telefone, mail, morada e data de nascimento.
